# WebVision

Сайт компании WebVision - профессиональная веб-разработка в Казахстане.

## О проекте

Этот репозиторий содержит исходный код корпоративного сайта компании WebVision, включая:
- Главную страницу
- Страницы портфолио
- Стили и скрипты

## Технологии

- HTML5
- CSS3
- JavaScript
- AOS (Animate On Scroll)
- Font Awesome

## Структура проекта

```
site/
├── assets/
│   ├── css/
│   │   ├── styles.css      # Основные стили
│   │   └── normalize.css   # CSS нормализация
│   ├── js/
│   │   └── script.js       # JavaScript функционал
│   └── images/             # Изображения и иконки
├── pages/
│   ├── portfolio1.html     # Страница портфолио 1
│   ├── portfolio2.html     # Страница портфолио 2
│   └── portfolio3.html     # Страница портфолио 3
└── index.html              # Главная страница
```

## Особенности

- Полностью адаптивный дизайн
- Современные анимации
- Оптимизированная производительность
- SEO-friendly структура
- Интеграция с WhatsApp
- Анимированные цены
- Интерактивное портфолио

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
