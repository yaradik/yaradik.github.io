Лендинг

О проекте

Продающий лендинг для компании, предоставляющей услуги по цифровому маркетингу. Проект включает в себя анимированные элементы, форму обратной связи и интеграцию с CRM-системой.

Реализованные функции

  • Адаптивный дизайн
  • Анимации при скролле
  • Форма обратной связи
  • Интеграция с CRM
  • Система онлайн-оплаты
  • Чат поддержки
  • Аналитика конверсий

Использованные технологии

  • HTML5, CSS3, JavaScript
  • GSAP для анимаций
  • PHP для обработки форм
  • MySQL для базы данных
  • Google Analytics
  • Facebook Pixel
