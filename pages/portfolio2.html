Интернет-магазин

О проекте

Полнофункциональный интернет-магазин с интеграцией с 1С и системой управления заказами. Проект включает в себя личный кабинет пользователя, систему скидок и бонусов, а также мобильное приложение.

Реализованные функции

  • Каталог товаров с фильтрацией
  • Корзина покупок
  • Система пользовательских аккаунтов
  • Онлайн-оплата
  • Интеграция с 1С
  • Система скидок
  • Мобильное приложение

Использованные технологии

  • Vue.js
  • Laravel
  • MySQL
  • Redis
  • WebSocket
  • REST API
